# Champion Mode

This document was created with the assistance of an AI.

## Role fantasy
You are embodied: you bleed, you negotiate, you build. Your power is real but bounded.

## Primary verbs
- Move / explore
- Gather / craft / build
- Assign labor and priorities
- Fight or evade threats
- Speak / persuade / intimidate
- Lead rituals (non-miracle) that prepare the settlement

## Constraints
- Physical proximity and time cost
- Limited information (fog-of-war, rumors)
- Social accountability (you can lose trust)
- Mortality (death/retirement possible)

## Champion systems (feature list)
1. **Settlement management**
   - building placement, zoning, upgrades
2. **Labor assignment**
   - jobs, schedules, emergency roles
3. **Expeditions**
   - scouting, relic recovery, gate reconnaissance
4. **Threat response**
   - defense planning, triage, evacuation
5. **Public leadership**
   - speeches, trials, councils, rites
6. **Witnessing**
   - champion presence increases “verification weight” for events

## Data needs (implementation-facing)
- Champion stats: health, stamina, charisma, insight, resilience
- Equipment/abilities: grounded “tech-ritual” gear
- Relationship graph with key NPCs
- Position + action log for witness weight

## UI notes
- Day HUD: needs alerts, job queue, map, threat banner
- “Witness markers” that tag major events as seen/unseen by credible observers
