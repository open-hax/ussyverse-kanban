# Miracles System

This document was created with the assistance of an AI.

## Definition
A **Miracle** is a deliberate rule-bend enacted during Deity phase (or by NPC gods) that produces a visible effect and a belief ripple.

## Miracle lifecycle
1. **Intent selection**
2. **Cost calculation**
3. **Casting / manifestation**
4. **Witnessing & evidence creation**
5. **Verification phase**
6. **Belief outcome**
7. **Fallout + counterplay**
8. **Cooldown / debt settlement**

## Miracle types (starter set)
- **Boon:** heal, empower, bountiful harvest, morale surge
- **Sign/Omen:** reveal information, mark a location, foreshadow threat
- **Constraint:** bind a creature, seal a rift temporarily
- **Transmutation:** alter resource type or building function
- **Rewrite:** rare; changes a world rule temporarily (high debt)

## Costs & currencies
- Influence (always)
- Devotion (optional booster)
- Material sacrifice (from settlement stores)
- Entropy Debt (accumulates; increases Chaos probability)
- Pantheon standing (some miracles offend domains)

## Verification (core mechanic)
Miracles must be “accepted” by the world to realize their full benefit.

### Verification inputs
- **Witness credibility:** champion present, respected NPCs, number of witnesses
- **Evidence artifacts:** lingering effects, relic traces, recorded omens
- **Narrative fit:** does it match the god’s domain and prior behavior?
- **Counterclaims:** rival gods can muddy belief with conflicting signs
- **Social context:** fear, hope, faction propaganda

### Verification outcomes
- **Confirmed:** full effect + devotion gain
- **Contested:** partial effect + social conflict
- **Debunked:** reduced effect + reputation loss + Chaos spike
- **Backfire:** inverted effect or Chaos manifestation

## Counterplay
- Rival gods cast “Doubt” miracles.
- Social systems generate skepticism or cult splits.
- Chaos can imitate miracles, making the world paranoid.

## UI requirements
- Miracle card: cost breakdown, predicted verification odds, predicted fallout
- Post-cast report: what was seen, who believes, who doubts, what changed
