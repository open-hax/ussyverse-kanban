# Colony Sim Systems

This document was created with the assistance of an AI.

## Settlement goals
- Create a living society with needs, culture, and conflict.
- Enable meaningful production and logistics without turning into spreadsheet play.

## Core loops
- Needs → work → resources → safety → devotion → miracles → chaos → needs

## Systems
### Buildings
- Housing, food production, workshops, storage
- Shrines/temples (devotion production)
- Wards/containment structures (reduce thinness)
- Council hall (social governance)
- Gate observatory (intel + risk)

### Resources (starter)
- Food (perishables)
- Water/energy (setting-appropriate: “fire” as resource)
- Materials (mud/stone/metal)
- Relics (rare, used for miracles/wards)
- Knowledge/omens (intangible, used for planning)

### Production
- Work orders and queues
- Skilled labor modifiers
- Disruptions from fear, disease, chaos events

### Logistics
- Stockpiles
- Priority routing (simple, readable)
- Emergency states (siege, famine)

## Governance
- Policies that alter behavior:
  - rationing
  - curfews
  - mandatory worship
  - exile/clemency
- Policies affect trust and verification.

## Edge cases
- Miracle-created abundance can collapse economy and belief.
- Too much repression triggers cults or sabotage.
