# Art Direction

This document was created with the assistance of an AI.

## Visual identity
**Glitch-mythic sci‑fi**:
- Marble + neon
- Temple-servers, holographic friezes
- Corrupted tape, scanlines, chromatic aberration as controlled accents
- Black butterflies with red/purple glitch glow
- Enormous white-light nodes at the Gates

## Readability rules
- Glitch is never allowed to obscure gameplay readability.
- Corruption effects must always be:
  - legible
  - attributable (you can tell what caused it)
  - and graded (levels 1–5)

## Character style
- Duck people are expressive, slightly surreal, but grounded enough for empathy.
- Gods appear as interfaces: masks, constellations, modular synth towers.

## UI styling
- “Corrupted parchment terminal”: mythic typography + system noise
- Distinct visual modes for Day vs Night
