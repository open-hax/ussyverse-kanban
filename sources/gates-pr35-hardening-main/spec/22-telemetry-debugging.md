# Telemetry, Debugging, and Replay

This document was created with the assistance of an AI.

## Non-negotiables
- Ability to answer: “what changed, and why?”

## Tools
- Timeline view of events with links to:
  - involved agents
  - resources
  - miracles
  - regions
- “Explain” tool for:
  - verification outcomes
  - agent decisions
  - chaos spawns

## Replay
- Chunked replay:
  - daily segments
  - incident clips
- Export/share: “myth recap” package (data only)
