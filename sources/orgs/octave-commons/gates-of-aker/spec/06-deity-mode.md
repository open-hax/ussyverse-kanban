# Deity Mode

This document was created with the assistance of an AI.

## Role fantasy
You are a force, not a body: you bend probability, interpret omens, and contend with other gods.

## Primary verbs
- Select and cast miracles (interventions)
- Allocate influence (domains, followers, locations)
- Negotiate with pantheon entities
- Shape beliefs via signs/omens
- Contain or redirect Chaos manifestations

## Constraints
- Resource costs: influence, devotion, sacrifice, entropy debt
- Jurisdiction: domain limits; some zones blocked by rival gods
- Verification: miracles are less effective if disbelieved or countered
- Blowback: Chaos escalates with rule-breaking

## Deity resources (suggested)
- **Influence:** spend to act; regenerates via belief/rituals
- **Devotion:** produced by settlement worship; boosts miracle potency
- **Entropy Debt:** accumulates from miracles; increases Chaos probability
- **Favor/Standing:** pantheon reputation; gates access and interdicts

## Night UI
- “Constellation board” of domains, followers, gates, omens
- Miracle deck builder / loadout
- Pantheon diplomacy view (alliances, grudges, interdictions)
