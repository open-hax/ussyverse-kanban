# UI/UX

This document was created with the assistance of an AI.

## Principles
- Surface causality: show why something happened.
- Keep the player oriented: clear phase shift cues.
- Let mythic language exist without hiding numbers.

## Key screens
### Day (Champion)
- World map + settlement overlay
- Task/labor manager
- Threat banner + incident log
- Relationship panel (top influencers)
- Gate status widget (thinness, containment, attention)

### Night (Deity)
- Constellation board (domains/regions/followers)
- Miracle deck/loadout
- Verification report center
- Pantheon diplomacy view
- Omen timeline (what the world is “telling” you)

## Notifications
- Tiered urgency: info / warning / crisis
- Corruption events use distinct audio + iconography

## Accessibility
- Glitch effects adjustable
- Colorblind-safe icons for corruption levels
