# Streaming & Community-Friendly Features

This document was created with the assistance of an AI.

## Goals
- Make emergent stories shareable.
- Keep “what happened” legible to viewers.

## Features
- **Witness Log recap**: daily summary cards (Day end and Night end)
- **Myth cards**: significant events framed as “myths” (with uncertainty if contested)
- **Audience mode** (optional): viewer-friendly overlay of:
  - gate thinness
  - devotion
  - current pantheon conflicts
  - miracle odds

## Ethical consideration
If viewer interaction ever affects gameplay, it must be:
- opt-in
- transparent
- bounded
- and never manipulative toward real-world actions.
