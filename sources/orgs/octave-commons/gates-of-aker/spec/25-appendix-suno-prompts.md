# Appendix — Music Prompt Seeds (Suno-style)

This document was created with the assistance of an AI.

## Theme prompt: Promethean glitchcore (system failing musically)
Style: Experimental electronic / dark synth / glitch ambient / industrial sci-fi.
90–105 BPM, half-time feel. Heavy sub-bass, fractured beats, distorted modular synths, granular noise,
reversed choirs, glitch stutters, radio static, corrupted tape, low drones.
Mood: uncanny, beautiful, destabilizing.
Structure: slow burn → collapse → strange calm.
Vocal: Minimal. Whispered phrases, chopped samples, pitch-shifted murmurs.
Lyrics themes: stolen fire, black butterflies, gates, memory, “verification” (belief vs truth), myth as data.

## Theme prompt: Duck discovering itself (AI sense-making)
Same palette, but emphasize:
- discovery, calibration, wonder through distortion
- tiny melodic motif that “learns” to stabilize against noise
- whispers: “I can see the sound / I can hear the light”
- chorus bursts only when a miracle is confirmed

## Palette prompt: Pandora event / chaos eruption
Increase granular noise and choir reversals.
Use “flutter” percussion that resembles wings.
Drop to near-silence after climax; end on strange calm drone.
